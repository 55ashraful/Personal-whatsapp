# ChatHub Setup Guide - বিস্তারিত নির্দেশনা

এই গাইড অনুসরণ করে আপনি সম্পূর্ণ ChatHub অ্যাপ্লিকেশন চালু করতে পারবেন।

## Step 1: Firebase Project তৈরি করুন

### 1.1 Firebase Console এ যান
- https://console.firebase.google.com এ ভিজিট করুন
- Google Account দিয়ে Login করুন

### 1.2 নতুন Project তৈরি করুন
```
1. "Create Project" বাটনে ক্লিক করুন
2. Project Name লিখুন: "ChatHub" বা যেকোনো নাম
3. Google Analytics: Optional (উপেক্ষা করতে পারেন)
4. "Create Project" ক্লিক করুন
5. প্রজেক্ট লোড হওয়ার জন্য অপেক্ষা করুন
```

### 1.3 Firebase Config পান
```\nবাম পাশের মেনু থেকে:\n1. \"Project Settings\" (গিয়ার আইকন) ক্লিক করুন\n2. \"General\" ট্যাবে থাকুন\n3. নীচে \"Your apps\" সেকশনে যান\n4. Web icon (</>)  ক্লিক করুন\n5. App nickname দিন: \"ChatHub Web\"\n6. \"Register app\" ক্লিক করুন\n7. Firebase config কপি করুন (পরবর্তীতে প্রয়োজন হবে)\n```\n\n## Step 2: Firebase Features সেটআপ করুন\n\n### 2.1 Authentication Enable করুন\n```\n1. বাম মেনু > Authentication ক্লিক করুন\n2. \"Get Started\" বাটন ক্লিক করুন\n3. \"Phone\" অপশন খুঁজে বের করুন\n4. Phone এর পাশের Edit করুন\n5. \"Enable\" এ ক্লিক করুন\n6. SMTP settings স্কিপ করুন (test mode এর জন্য)\n7. \"Save\" করুন\n\nTest Phone Numbers যোগ করুন:\n1. Phone Authentication এ \"Phone numbers for testing\" সেকশন থাকবে\n2. \"Add phone number\" ক্লিক করুন\n3. Phone number লিখুন: +880-1234567890 (আপনার দেশের কোড)\n4. Test OTP দিন: 123456 (যেকোনো ৬ ডিজিট)\n5. \"Add\" করুন\n\nএরকম আরও ২-৩টি test number যোগ করুন\n```\n\n### 2.2 Firestore Database তৈরি করুন\n```\n1. বাম মেনু > Firestore Database ক্লিক করুন\n2. \"Create database\" বাটন ক্লিক করুন\n3. Test mode select করুন (development এর জন্য)\n4. Location select করুন (আপনার কাছাকাছি)\n5. \"Enable\" করুন\n6. Database লোড হওয়ার জন্য অপেক্ষা করুন\n```\n\n### 2.3 Firestore Security Rules সেটআপ করুন\n```\n1. Firestore Database > Rules ট্যাবে যান\n2. সমস্ত content মুছে ফেলুন\n3. এই code পেস্ট করুন (firestore.rules ফাইল থেকে):\n\n  rules_version = '2';\n  service cloud.firestore {\n    match /databases/{database}/documents {\n      match /users/{userId} {\n        allow read, write: if request.auth.uid == userId;\n      }\n      match /chats/{chatId} {\n        allow read, write: if request.auth != null && \n          request.auth.uid in resource.data.participants;\n      }\n      match /groups/{groupId} {\n        allow read, write: if request.auth != null && \n          request.auth.uid in resource.data.members;\n      }\n      match /{document=**} {\n        allow read, write: if request.auth != null;\n      }\n    }\n  }\n\n4. \"Publish\" করুন\n```\n\n### 2.4 Storage সেটআপ করুন\n```\n1. বাম মেনু > Storage ক্লিক করুন\n2. \"Get Started\" করুন\n3. Test mode select করুন\n4. Location select করুন\n5. \"Done\" করুন\n```\n\n## Step 3: Repository Setup করুন\n\n### 3.1 Repository Clone করুন\n```bash\n# Terminal/CMD তে যান এবং এই কমান্ড চালান:\ngit clone https://github.com/55ashraful/personal-whatsapp.git\ncd personal-whatsapp\n```\n\n### 3.2 Firebase Config আপডেট করুন\n```\n1. firebase.js ফাইল খুলুন\n2. এই লাইন খুঁজে বের করুন:\n   const firebaseConfig = { ... }\n\n3. Step 1.3 এ পাওয়া config দিয়ে প্রতিস্থাপন করুন\n\n4. ফাইল save করুন\n```\n\n### 3.3 index.html সেটআপ করুন\n```\n1. index.html ফাইল খুলুন\n2. <head> এর শেষে এই লাইন খুঁজুন:\n   <link rel=\"manifest\" href=\"manifest.json\">\n\n3. নিচে এই লাইন যোগ করুন:\n   <script src=\"firebase.js\"></script>\n   <script src=\"database.js\"></script>\n   <script src=\"chat.js\"></script>\n   <script src=\"group.js\"></script>\n   <script src=\"call.js\"></script>\n   <script src=\"profile.js\"></script>\n   <script src=\"app.js\"></script>\n```\n\n## Step 4: Local Testing (আপনার কম্পিউটারে)\n\n### 4.1 Simple HTTP Server চালান\n\n**Option 1: Python ব্যবহার করে**\n```bash\n# Python 3 ইনস্টল থাকলে:\ncd personal-whatsapp\npython -m http.server 8000\n```\n\n**Option 2: Node.js ব্যবহার করে**\n```bash\n# Node.js ইনস্টল থাকলে:\ncd personal-whatsapp\nnpx http-server\n```\n\n**Option 3: VS Code Live Server**\n```\n1. VS Code এ project খুলুন\n2. Extensions এ \"Live Server\" ইনস্টল করুন\n3. index.html এ right-click করুন\n4. \"Open with Live Server\" ক্লিক করুন\n```\n\n### 4.2 Browser এ Test করুন\n```\n1. Browser খুলুন এবং এই address এ যান:\n   http://localhost:8000\n\n2. আপনার test phone number দিয়ে login করুন\n\n3. Test OTP দিন (যা Firebase এ সেট করেছেন)\n\n4. Profile complete করুন\n\n5. দ্বিতীয় browser/window এ দ্বিতীয় phone number দিয়ে login করুন\n\n6. একে অপরের সাথে chat করুন এবং test করুন\n```\n\n## Step 5: Production Deployment\n\n### 5.1 Vercel এ Deploy করুন (সহজ পদ্ধতি)\n\n```bash\n# 1. Vercel CLI ইনস্টল করুন\nnpm i -g vercel\n\n# 2. Vercel এ login করুন\nvercel login\n\n# 3. Project deploy করুন\ncd personal-whatsapp\nvercel\n\n# 4. Prompts অনুসরণ করুন:\n# - \"Set up and deploy?\" - Yes\n# - \"Which scope do you want to deploy to?\" - আপনার account\n# - \"Found project.json. Link it?\" - No\n# - \"Project name\" - chathub-app\n# - \"Directory to deploy\" - . (current)\n\n# 5. Deploy হয়ে গেলে URL পাবেন\n```\n\n### 5.2 Firebase Hosting এ Deploy করুন (Alternative)\n\n```bash\n# 1. Firebase CLI ইনস্টল করুন\nnpm i -g firebase-tools\n\n# 2. Firebase এ login করুন\nfirebase login\n\n# 3. Project initialize করুন\ncd personal-whatsapp\nfirebase init\n\n# Prompts:\n# - \"Which features to enable?\" - Hosting\n# - \"Project to associate\" - আপনার Firebase project\n# - \"Public directory\" - .\n# - \"Configure as single-page app?\" - Yes\n# - \"Set up automatic builds?\" - No\n\n# 4. Deploy করুন\nfirebase deploy --only hosting\n\n# 5. আপনার Firebase Hosting URL পাবেন\n```\n\n## Step 6: Custom Domain সেটআপ (Optional)\n\n### 6.1 Vercel এ Domain যোগ করুন\n```\n1. Vercel Dashboard এ project খুলুন\n2. \"Settings\" > \"Domains\" এ যান\n3. আপনার domain name দিন\n4. DNS settings নোট করুন\n5. আপনার domain provider এ DNS update করুন\n6. Verify হওয়ার জন্য অপেক্ষা করুন (15-60 মিনিট)\n```\n\n## Step 7: Security Configuration\n\n### 7.1 CORS সেটআপ করুন\n```bash\n# Firebase CLI দিয়ে CORS setting করুন:\ngsutil cors set cors.json gs://your-project.appspot.com\n\n# cors.json ফাইল তৈরি করুন:\n[\n  {\n    \"origin\": [\"https://yourdomain.com\"],\n    \"method\": [\"GET\", \"PUT\", \"POST\", \"DELETE\"],\n    \"responseHeader\": [\"Content-Type\"],\n    \"maxAgeSeconds\": 3600\n  }\n]\n```\n\n### 7.2 Environment Configuration\n```bash\n# .env ফাইল তৈরি করুন (production):\nFIREBASE_API_KEY=your_api_key\nFIREBASE_AUTH_DOMAIN=your_auth_domain\nFIREBASE_PROJECT_ID=your_project_id\nFIREBASE_STORAGE_BUCKET=your_storage_bucket\nFIREBASE_MESSAGING_SENDER_ID=your_messaging_id\nFIREBASE_APP_ID=your_app_id\n```\n\n## Step 8: Monitoring এবং Maintenance\n\n### 8.1 Firebase Console Monitor করুন\n```\nনিয়মিত চেক করুন:\n1. Authentication > Users (active users count)\n2. Firestore > Usage (read/write operations)\n3. Storage > Usage (storage consumption)\n4. Analytics > Dashboard\n```\n\n### 8.2 Errors Track করুন\n```\n1. Browser console (F12) check করুন\n2. Firebase Console > Realtime Database > Usage tab\n3. CloudFunctions logs (যদি থাকে)\n```\n\n## Troubleshooting\n\n### সমস্যা: OTP পাচ্ছি না\n```\nসমাধান:\n1. Firebase test number ব্যবহার করছেন কিনা check করুন\n2. Firebase Console > Authentication > Phone এ test number সঠিক আছে কিনা দেখুন\n3. Browser console এ error আছে কিনা check করুন\n```\n\n### সমস্যা: Messages sync না হওয়া\n```\nসমাধান:\n1. Firestore rules সঠিক আছে কিনা verify করুন\n2. Network connection check করুন\n3. Firestore Database > Data এ manually কিছু data রয়েছে কিনা দেখুন\n4. Browser console এ error check করুন\n```\n\n### সমস্যা: Call feature কাজ না করা\n```\nসমাধান:\n1. Camera/Microphone permission দিয়েছেন কিনা check করুন\n2. Browser console এ WebRTC errors আছে কিনা দেখুন\n3. Internet connection stable আছে কিনা check করুন\n4. Browser এর settings এ microphone/camera enable করুন\n```\n\n## Performance Optimization\n\n### 8.1 Database Optimization\n```\n1. Composite indexes তৈরি করুন (বড় queries এর জন্য)\n2. Sharding ব্যবহার করুন (বড় collections এর জন্য)\n3. Batch operations করুন (multiple writes এর জন্য)\n```\n\n### 8.2 Frontend Optimization\n```\n1. CSS minify করুন\n2. JavaScript bundle করুন\n3. Images optimize করুন\n4. Lazy loading implement করুন\n```\n\n## Scaling Guidelines\n\nযখন আরও বেশি users যুক্ত হবেন:\n\n```\n1. Firebase Plan upgrade করুন (Blaze plan)\n2. CDN setup করুন (Cloudflare etc)\n```\n\n## Final Checklist\n\n```\n✅ Firebase project তৈরি করেছেন\n✅ Authentication configure করেছেন\n✅ Firestore database তৈরি করেছেন\n✅ Storage enable করেছেন\n✅ Firebase config আপডেট করেছেন\n✅ Local server এ test করেছেন\n✅ অন্তত 2টি account দিয়ে verify করেছেন\n✅ Production এ deploy করেছেন\n✅ Custom domain সেটআপ করেছেন (Optional)\n✅ Security rules set করেছেন\n✅ Monitoring setup করেছেন\n```\n\n## সাহায্যের জন্য\n\n- Firebase Documentation: https://firebase.google.com/docs\n- Project GitHub: https://github.com/55ashraful/personal-whatsapp\n- Email: ashrafulhaqedulon@gmail.com\n\n---\n\n**Happy deploying! 🚀**\n
